# Tugas 15 - Working with Clouds

Project ini merupakan aplikasi web sederhana untuk menguji konsep cloud scaling pada saat terdapat banyak request dari user.

## Teknologi yang Digunakan

- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- GitHub
- Azure App Service

## Fitur Aplikasi

- Cek status server
- Simulasi 50 request
- Menampilkan total request
- Menampilkan load request
- Menampilkan durasi pengujian
- Menampilkan response API

## Cara Menjalankan Aplikasi

```bash
npm install
npm start
